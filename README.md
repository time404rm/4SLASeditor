# 4SLASeditor

**Лёгкий WYSIWYG-редактор на чистом JavaScript. 1383 строки, 0 зависимостей.**

Встроенный редактор [4SLAS CMS](https://github.com/time404rm/4SLAS-cms) —
бесплатной, самодостаточной и безопасной CMS для блога на PHP 8.

Сайт проекта: https://time404.ru

---

## Возможности

**Текстовое форматирование:**
- Жирный, курсив, подчёркнутый, зачёркнутый
- Цвет текста (foreColor) и фона (backColor) — выбор из палитры
- Заголовки H2, H3
- Цитаты (blockquote)
- Форматированный текст (pre)

**Списки и выравнивание:**
- Маркированный (ul) и нумерованный (ol) списки
- Выравнивание: влево, по центру, вправо, по ширине

**Медиа и ссылки:**
- Загрузка изображений (POST /api/upload)
- Вставка URL изображения
- Загрузка файлов (POST /api/upload)
- Вставка видео (YouTube, VK, плеер)
- Вставка ссылок и якорей
- Таблицы

**Код и специальное:**
- Вставка кода с подсветкой синтаксиса — 15 языков: PHP, JavaScript, HTML, CSS, SQL, Python, Java, C++, C#, Go, Ruby, JSON, XML, Bash, Markdown
- Режим HTML — переключение между WYSIWYG и исходным кодом
- Эмодзи — встроенная палитра смайлов
- Горизонтальная линия

**Управление и UX:**
- Отменить / Повторить (Ctrl+Z / Ctrl+Y)
- Полноэкранный режим
- Счётчик слов и символов в статус-баре
- Индикатор режима (WYSIWYG / HTML)

**Интеллектуальные функции:**
- Drag-and-drop изображений из проводника прямо в редактор
- Paste-очистка — удаляет мусорное форматирование из Word и браузера
- Tab-отступы при редактировании кода
- Автоматическая синхронизация содержимого с hidden textarea
- Инлайн-подсветка синтаксиса — без Highlight.js, собственные регулярки
- Подсветка активных кнопок тулбара в реальном времени

```html
</script>
```
